# 2026-04-24 工作日志

## 1. 修复网页无法访问问题
- Nginx 服务停止，原因可能是服务器重启后未自动启动
- 用户在阿里云控制台重置了 root 密码
- 执行 systemctl start nginx + systemctl enable nginx 修复
- 设置了开机自启，防止后续再出现此问题

## 2. 修复数据不保存的 bug
- 问题：saveStoreField() 函数只更新内存，不保存到服务器/localStorage
- 修复：在 saveStoreField() 中添加 saveStoresToServer() 和 localStorage 保存

## 3. 添加多设备数据轮询同步
- 每 5 秒从服务器拉取最新数据
- 本地正在保存时暂停轮询，避免数据覆盖
- 登出时停止轮询，登录时启动
- 基于 hash 比对判断数据是否变化，避免无意义的页面重渲染

## 4. 部署到服务器
- 通过 scp 将修改后的 HTML 文件上传到服务器
- 服务器路径: /var/www/store-progress-manager/进度管理系统.html
- 已验证文件更新成功
